# Dyna

Dyna is a graphql server that uses Drizzle schemas to generate the graphql schema and resolvers.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/download/) (version 16.0.0 or higher)

### Installation

```bash
pnpm install
```

### Running the dev server

```bash
pnpm dev
```

### Building the app

```bash
pnpm build
```
